## üîç **Analysis Algorithm**

### **Performance Score Calculation:**
- **Base Score:** 100%
- **Deductions for:**
  - Long execution times (>2s, >5s, >10s)
  - High CPU usage (>2s, >5s)  
  - Excessive SOQL queries (>20, >50, >100)
  - High DML operations (>100, >150)

### **Risk Score Assessment:**
- **Execution Time Risk** - Delays in user experience
- **CPU Usage Risk** - Resource consumption impact
- **Query Volume Risk** - Governor limit concerns
- **DML Volume Risk** - Bulk processing efficiency
